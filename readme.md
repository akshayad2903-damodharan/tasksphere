# 🚀 TASKSPHERE - Personal Task Management Dashboard

A modern, full-stack personal productivity web application built with **Python Flask**, **SQLite**, **HTML/CSS/JavaScript**.

## ✨ Features
- ✅ Secure User Authentication (Signup/Login/Logout)
- ✅ Modern Dashboard with Statistics Cards
- ✅ Task Management (Add/Edit/Delete/Complete)
- ✅ Student Profile Management
- ✅ Responsive Professional UI
- ✅ Progress Tracking & Recent Activity
- ✅ Sidebar Navigation
